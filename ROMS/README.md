# ROM Files

This directory is for Oric ROM files required by the emulator.

## Required ROMs

The following ROM files are **not included** in this repository and must be obtained separately:

| Filename | Description | Size |
|----------|-------------|------|
| `basic10.rom` | Oric 1 BASIC ROM | 16 KB |
| `basic11b.rom` | Oric Atmos BASIC ROM | 16 KB |
| `microdis.rom` | Microdisc controller ROM | 8 KB |

## Obtaining ROMs Legally

You must obtain these ROM files legally. Options include:

1. **Dump from original hardware** - If you own an Oric computer, you can dump the ROMs yourself
2. **Oric preservation sites** - Check legitimate preservation sites like [oric.org](https://oric.org)
3. **Other emulators** - Some Oric emulators may bundle ROMs with permission

## Place Files Here

Once obtained, place the ROM files directly in this `ROMS/` directory. The emulator will look for them here by default (configurable in `auric.yaml`).
